# Followers Check

A simple HTML page that displays the total count of Instagram followers from a JSON file.

## Files

- `followers.json` - Your Instagram followers data
- `generate-html.js` - Script that generates the HTML file
- `index.html` - The generated HTML page (auto-generated)

## Usage

1. Make sure you have `followers.json` in the project directory
2. Run the generator script:
   ```bash
   node generate-html.js
   ```
3. Open `index.html` in your browser to see the follower count
